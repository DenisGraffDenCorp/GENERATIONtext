# GENERATIONtext
Постройте генератор текстов, который на вход получает текст на русском языке и далее продолжает его. Для генерации нового символа последовательности нужно использовать цепи Маркова, вероятности появления символа нужно взять из задания 2.


GEN.py - с помощью команды python GEN.py markov.db 500 генерирурет текст
SQLDB - создает таблицу БД
